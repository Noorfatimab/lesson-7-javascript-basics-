# Lesson 7 – JavaScript Basics

This project introduces JavaScript for the first time.

## What this project shows
- How to connect a JavaScript file to an HTML file
- How JavaScript runs in the browser
- How JavaScript can communicate with the user using an alert

## Files included
- index.html – The structure of the webpage
- script.js – The JavaScript logic
- README.md – Project explanation

This is the first step toward making webpages interactive.
